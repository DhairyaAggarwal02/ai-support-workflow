# Account Access Policy

## Password Reset
Users can reset their password using the password reset link on the login page.

## Account Lockout
Accounts may be temporarily locked after multiple failed login attempts.

## Two-Factor Authentication
Users with two-factor authentication enabled must verify their identity using their configured authentication method.

## Email Change
Requests to change the account email address should be verified before changes are made.

## Account Ownership Disputes
Account ownership disputes should be escalated to a human support agent.

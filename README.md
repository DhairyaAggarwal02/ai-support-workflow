# AI Support Workflow Automation System

This project is an agentic AI workflow automation system for customer support and business operations.

It is designed to:
- classify customer intents
- route queries to the correct domain
- retrieve answers from a knowledge base
- detect risky or sensitive queries
- decide whether to auto-respond or escalate to a human
- estimate business value through time savings and automation metrics

## Goal

Build a reusable AI support system that can work across multiple business domains by swapping the knowledge base.
